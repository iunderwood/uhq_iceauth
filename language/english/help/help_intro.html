Help: UHQ_IceAuth

Introduction


This module provides a URL authentication facility to IceCast servers running 2.3.2 and higher.


The feature functionality is achieved in a client/server manner where requests originating from an IceCast server are checked against a defined set of mount points. In addition to allowing or disallowing a streaming session, the module is capable of pushing out intro files as well as imposing time limits on the stream before a client needs to connect again.

In addition to authentication, the module is capable of performing a rudimentary set of accounting. This provides a basic level of accounting for TTSL (total time spent listening), as well as the user agents which connect into the system.
